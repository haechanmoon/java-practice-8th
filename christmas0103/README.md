# 미션 - 크리스마스 프로모션

version 1.0 초안 작성

## 기능 구현 리스트

- **Menu(enum)**
- [x] 자신의 메뉴를 반환
- [x] 메뉴의 타입 반환

- **Calendar(enum)**
- [x] 1~25면 D-day할인
- [x] 평일/ 주말 체크
- [x] 특별할인 체크

- **Calculator**
- [ ] 평일 주말 할인
- [ ] 별 할인
- [ ] 샴페인 여부 확인

- **Badge**
- [ ] 자신의 맞는 혜택 받음

- **Order**
- [ ] 주문 메뉴와 개수 저장
- [x] 디저트가 몇개 있는지 반환
- [x] 메인이 몇개 있는지 반환
- [ ] 할인 전 주문금액 계산
- [ ] (검증) 음료만 받았는지
- [ ] (검증) 메뉴와 개수가 20개 초과했는지

**실패**
돌어가도록 짜보자.
