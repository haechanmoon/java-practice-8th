# java-lotto-precourse

version 1.0 초안 작성

## 기능 구현 리스트

#### domain

- **Lotto**
- [x] 자신과 당첨번호를 비교 후 몇개인지 반환
- [x] 보너스 번호를 갖고있는지 반환
- [x] 자신의 번호를 문자열로 반환
- [ ] (검증) 자신의 번호가 6개인지
- [ ] (검증) 자신의 번호가 중복되진 않았는지
- [ ] (검증) 자신의 번호의 범위가 1-45가 맞는지

- **Lottos**
- [x] 로또들의 번호를 문자열로 반환

- **WinningNumbers**
- [x] 당첨번호를 로또번호와 비교
- [x] 당첨번호를 보너스번호와 비교
- [x] 당첨번호 매칭 등수가 잘 나오는지
- [ ] (검증) 자신의 번호가 6개인지
- [ ] (검증) 자신의 번호가 중복되진 않았는지
- [ ] (검증) 자신의 번호의 범위가 1-45가 맞는지

- **Parser**
- [x] 입력된 문자를 숫자로 변환
- [x] 입력된 문자를 List<Integer> 로 변환
- [ ] (검증) 입력된 문자가 빈값은 아닌지
- [ ] (검증) 입력된 문자가 숫자형태인지
- [ ] (검증) 입력된 문자가 1000단위인지

- **Rank**
- [x] 로또번호가 당첨번호/보너스번호와 비교하여 몇등인지 반환

- **RandomGenerator**
- [x] 1-45 번호를 6개 생성
