# java-lotto-precourse


v1.0 초안 작성
## 기능 구현 리스트

- **Lotto**
- [x] (검증) 자신의 번호가 6개인지
- [x] (검증) 번호의 범위가 1~45인지
- [x] (검증) 번호가 중복되진 않는지
- [x] 자신의 번호를 반환
- [x] 자신의 번호와 당첨번호가 몇개 매칭되는지 반환
- [x] 자신의 번호와 보너스번호 매칭 여부 반환

- **Lottos**
- [x] 로또들의 번호를 반환
- [x] Map<Rank,Integer> 초기 생성
- [x] winningNumbers와 비교하여 Map<Rank,Integer>생성

- **RandomNumberGenerator**
- [x] 1~45의 중복되지 않는 숫자 6개 반환

- **LottoStore**
- [x] 받은 액수만큼 로또 생성
- [x] (검증) 받은 돈이 1000원단위인지

- **Rank**
- [x] 꽝부터 1등까지의 등수 반환

- **Statistics**
- [ ] Map이용하여 수익률 계산

- **WinningNumbers**
- [x] (검증) 당첨 번호가 1~45의 수인지
- [x] (검증) 당첨 번호가 중복되진 않는지
- [ ] (검증) 당첨 번호가 6개인지
- [ ] (검증) 보너스 번호가 중복되진 않는지
- [x] 로또번호와 당첨번호의 매칭 개수 반환
- [x] 로또번호와 보너스번호의 매칭 개수 반환

## 테스트 리스트

- **Lotto**
- [x] 자신의 번호와 당첨번호가 잘 매칭 되는지
- [x] 자신의 번호와 보너스번호가 잘 매칭 되는지
- [x] (예외) 번호가 6개가 아닐 때
- [x] (예외) 번호가 범위를 넘어설 때
- [x] (예외) 중복된 번호가 있을 때

- **WinningNumbers**
- [ ] (예외) 당첨번호가 중복되었을 때
- [ ] (예외) 당첨번호가 범위를 벗어났을 때
- [ ] (예외) 보너스 번호와 당첨번호가 중복되었을 때
- [ ] 로또 번호와 당첨번호가 잘 매칭되는지
- [ ] 로또 번호와 보너스 번호가 잘 매칭되는지

- **Statistics**
- [ ] 결과가 수익률로 잘 나오는지
