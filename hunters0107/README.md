# java-hunters

## 기능 구현 리스트

- **RandomNumberGenerator**
- [x] 랜덤 번호를 생성

- **Hunter**
- [x] 한 라운드 전진
- [x] (검증) 이름이 5자 이하
- [x] 자신의 상태가 0이하이면 사망
- [x] 라운드 별 자신의 상태 반환

- **Hunters**
- [x] 라운드 별 Hunters의 상태 반환
- [x] 헌터들의 총 전리품 반환
- [x] 생존자별 전리품 반환
- [x] 사망자 반환
- [x] 라운드 별 사망자 제외 전진

- **Round(`enum`)**
- [x] 자신의 라운드 반환

- **HunterFactory**
- [x] 헌터들을 생성
