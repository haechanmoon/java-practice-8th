# 미션 - 숫자 야구

version 1.0 : 초안 작성

## 기능 구현 리스트

- **RandomGenerator**
- [ ] 겹치지 않는 List<Integer> 랜덤값 생성 

- **Result**
- [ ] 몇 스트라이크, 몇 볼인지 문자로 만들기
- [ ] 3스트라이크면 게임 마무리
- [ ] 게임을 재시작 할지 말지 정하기

- **Computer**
- [ ] 받은 값을 자신의 List<Integer> 와 비교 후 Result로 리턴

- **Parser**
- [ ] 받은 문자열을 숫자로 변환
- [ ] (검증) 입력값 숫자형태인지
- [ ] (검증) 입력값이 3개인지
- [ ] (검증) 재시작 입력이 1과 2중 하나인지.
- [ ] (검증) 숫자가 겹치지 않는지


## 테스트 리스트

- **Result**
- [ ] 3스트라이크일 때 문자열 확인
- [ ] 아무것도 없을 때 낫싱을 잘 출력하는지

- **Computer**
- [ ] 1스트라이크 1볼인 매치가 왔을 때 잘 출력되는지.
- [ ] 세개 다 맞았을 때 3스트라이크 나오는지
- [ ] 하나도 없을 때 낫싱을 잘 출력하는지.

- **Parser**
- [ ] (예외) 입력값이 숫자형태가 아닐 때
- [ ] (예외) 입력값이 4개일 때
- [ ] (예외) 재시작 입력이 1과 2중 가 아닐 때
- [ ] (예외) 숫자가 겹칠 때
