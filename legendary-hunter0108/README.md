# java-legendary-hunter

v1.0 초안 작성

## 기능 구현 리스트
- **Hunter**
- [ ] 갖고있는 돈을 반환한다.
- [ ] 전진한다.

- **Dungeons**
- [ ] 각 던전들에서 얻은 데이터를 Map형태로 반환한다.

- **RandomGenerator**
- [ ] 랜덤값 주입

- **Parser**
- [ ] 돈을 int로 반환

- **DungeonGenerator**
- [ ] 던전을 생성한다.

- **Calculator**
- [ ] 수익률을 계산한다.

- **Dungeon**
- [ ] 자신이 만난 던전을 반환한다.

## 테스트 리스트
- **Hunter**
- [ ] 슬라임 만났을 때 돈 확인

- **DungeonGenerator**
- [ ] 받은 액수에 맞게 던전이 생성되는지
- [ ] (예외) 돈의 단위가 1000원단위가 아닐 때

- **Dungeon**
- [ ] 자신이 맞는 던전을 구했는지
