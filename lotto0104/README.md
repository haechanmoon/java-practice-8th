# java-lotto-precourse

## 기능 구현 리스트

- **Lotto**
- [ ] (검증) 숫자 범위 1~45
- [ ] (검증) 숫자가 중복되진 않는지
- [ ] 당첨번호와 몇개가 맞았는지
- [ ] 보너스 번호가 맞았는지

- **WinningNumbers**
- [ ] (검증) 숫자 범위 1~45
- [ ] (검증) 숫자가 중복되진 않는지
- [ ] (검증) 보너스 번호가 중복되진 않는지
- [ ] Map활용 Rank와 개수를 반환

- **Rank**
- [ ] 자신의 랭크 반환

- **LottoStore**
- [ ] 받은 숫자만큼 로또 발행 (가격) (Lottos)

- **Lottos**
- [ ] 로또들의 맞은 개수 반환 (int)
- [ ] 보너스 번호 맞았는지 반환 (boolean)

- **RandomGenerator**
- [ ] 6개의 무작위 숫자 추출
