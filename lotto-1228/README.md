# java-lotto-precourse

> 1등을 잡아라! 로또 입니다. 금액을 입력한 후 , 당첨번호/보너스번호를 입력하면 당첨된 결과와 함께 수익률이 출력됩니다.

version = 1.7 : Parser 검증기능 추가
